# WELCOMEMESSAGES
### Add a custom messsage send by server to a channel.

### ADD to config file

```
channel-welcome {
    "#channel1" "Welcome to #channel1! Have a nice chatting!";
    "#channel2" "Welcome to #channel2! Have a great time!";
    "#channel3" "Welcome to #channel3! Feel at home.";
};

```
